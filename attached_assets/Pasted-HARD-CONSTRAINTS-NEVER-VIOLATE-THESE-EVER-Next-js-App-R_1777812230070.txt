E2 — Optimistic updates on mutations
For these high-frequency mutations, add optimistic updates:
- Mark as read (messages)
- Delete item from list (any list page)
- Toggle fee payment status

Pattern:
  onMutate: async () => {
    await queryClient.cancelQueries({ queryKey })
    const prev = queryClient.getQueryData(queryKey)
    queryClient.setQueryData(queryKey, optimisticValue)
    return { prev }
  }
  onError: (err, vars, ctx) => queryClient.setQueryData(queryKey, ctx.prev)
  onSettled: () => queryClient.invalidateQueries({ queryKey })
